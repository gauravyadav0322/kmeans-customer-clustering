
# Customer Segmentation Using KMeans Clustering

This project applies **KMeans Clustering** to segment customers based on their behaviour and attributes. The goal is to identify meaningful customer groups that help in personalised marketing, profiling, and better business decisions.

---

##  Project Structure
- `customer segmentation.ipynb`
- `Kmeans Clustering.ipynb`

---

##  Key Features
- Data cleaning and preprocessing  
- Feature selection and scaling  
- KMeans clustering using Elbow Method  
- Cluster visualisation with scatter plots  
- Insights and interpretation of each cluster  

---

## Technologies Used
- Python  
- NumPy  
- Pandas  
- scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook
